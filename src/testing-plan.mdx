## Format Variations

- [ ] **M3U8 Format:** Test videos that use M3U8 streaming format
- [ ] **WebM Format:** Test videos that use WebM container format
- [ ] **MP4 Format:** Test videos that use MP4 container format
- [ ] **MPD Format:** Test videos that use DASH MPD format

## Subtitle Tests

- [ ] **JSON Subtitles Only:** Videos with only JSON subtitle format
- [ ] **VTT Subtitles Only:** Videos with only VTT subtitle format
- [ ] **Multiple Subtitle Formats:** Videos with both VTT and JSON subtitles
- [ ] **No Subtitles:** Videos without any subtitle tracks

## Third Party Video Embeds

- [ ] **Publicly Available Embed:** Test Loom videos embedded on external websites
- [ ] **Private/Restricted Embed:** Test password-protected or restricted Loom embeds on external sites

## Error Handling

- [ ] **Invalid Video ID:** Test with non-existent Loom video IDs
- [ ] **Expired/Deleted Videos:** Test with videos that have been removed
- [ ] **Network Timeout:** Test behavior with slow/failed network connections
- [ ] **Empty JSON Response:** Test transcoded-url endpoint returning empty JSON
